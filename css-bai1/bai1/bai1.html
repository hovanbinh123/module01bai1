What is CSS?

Cascading Style Sheets (CSS) is used to format the layout if webpage.

with CSS,ypu can control the collor,font,the size of text,the spacing between elements,how elements are positioned and laid out,what background colors are to be used, different displays for different devices and screen sizes, and much more!

Tip: the word cascadingmeans that a style applied to a parent element will also apply to all children elements within the parent. so, if you set the color of the body text to "blue",all headings, paragraphs, and other text elements within the dofy will also get the same color (unless you specify something else)!
